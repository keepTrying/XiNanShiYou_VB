Control Notes

When opening a toolbar in the configuration utility that was saved in a previous build of the control, the toolbars will be shifted to the right or down. You will have to move the toolbars back to the correct position before saving.

If you are experiencing problems using the help system under VB 5.0 you will need to apply the VB 5.0 service pack 2 or greater to fix it.

For more detailed information, consult the Known Control Issues section of the help file.

Included Samples

In the Samples sub-folder of your ActiveListBar installation folder are various samples demonstrating the use of the ActiveListBar control. Brief descriptions of the samples can be found in the file Samples Descriptions.txt, located in the Samples folder.

System Requirements:
* A suitably-equipped computer running a 32-bit Windows operating system (Windows 95,
Windows NT 4.0 or higher). A minimum configuration of 16MB of RAM and a 256-color video
adapter is recommended.

* Approximately 8 megabytes of available disk space for a full installation

* A 32-bit development platform capable of hosting ActiveX cotnrols, such as a 32-bit version of
Visual Basic. All features may not be available on all platforms.
